## Starting and running the app

1. Build and run docker image:

```
npm run docker:build
npm run docker:run
```

2. Run and compose docker image:
   `npm run docker:compose`

3. Run in dev mode:
   `npm run dev`

4. Run and build the app:

````npm run build
npm start```
````

## Restart Docker

killall Docker && open /Applications/Docker.app
